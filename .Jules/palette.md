## 2025-05-14 - Confirmation for destructive actions
**Learning:** Destructive actions without confirmation (like task deletion) can lead to accidental data loss and user frustration. Adding a Material `AlertDialog` with a clear warning and styling the confirm button with the theme's error color provides a safe and intuitive user experience.
**Action:** Always implement a confirmation dialog for any destructive action and ensure icon-only buttons have descriptive tooltips for accessibility.
